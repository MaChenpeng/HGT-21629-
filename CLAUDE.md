# HGT21629管架标准图汇料工具

## 项目概述

本项目是将原始Excel VBA程序转换为Python实现的管架标准图物料汇总工具。用于根据HGT21629标准图集中的管架编号自动计算所需零件（槽钢、角钢、钢板、螺栓等）的设计量，并生成汇总表和明细表。

原始Excel系统包含两个工作簿：
- **输入和计算书**：用于输入管架编号并展开零件明细
- **支架汇总表**：用于汇总相同零件并匹配材料库

## 核心工作流程

### 1. 管架编号解析 (parser.py)

将管架编号字符串解析为结构化数据。不同型号有不同格式：

```
A1-A-200-C1     → 型号A1, 类型A, 管径200mm, 材质C1
D4-2-D-1000-200 → 型号D4, 数量2, 子项D, 长度1000mm, 高度200mm
F2-A1-200-C1    → 型号F2, 类型A1, 管径200mm, 材质C1
Q3-4-A-200-25-1024-500 → 型号Q3, 数量4, 子项A, 主管径200, 支管径25, 长度1024, 高度500
```

### 2. 零件计算 (calculator.py)

根据解析后的编号计算各零件的设计量。计算公式因型号而异：

**通用公式：**
- 长度类零件（槽钢、角钢等）：`设计量 = 长度(mm) × 数量 / 1000 → m`
- 面积类零件（钢板等）：`设计量 = 宽度(mm) × 高度(mm) × 数量 / 1000000 → m²`

**特殊型号公式示例：**
- **D6**: 第一根槽钢=高度, 第二根槽钢=长度×√2
- **D7-D10**: 总长度 = (长度 + 高度)
- **Q3**: 总长度 = (高度+长度+管径+支管径) × 1.45 × 数量 / 1000
- **D17**: H型钢 = (长度+高度+长度2+高度2) × 1.193 × 数量 / 1000

### 3. 汇总计算 (summarizer.py)

按(名称+规格+等级+材质)分组汇总设计量，并匹配材料库填充单重和总重。

**材料库匹配键：** `(名称及编码, 规格, 材质)`
**汇总分组键：** `(名称, 规格, 等级, 材质)`

### 4. Excel导出 (exporter.py)

导出三个工作表：
- **输入表**：原始管架编号和数量
- **明细表**：展开的零件明细
- **汇总表**：按材质分组的汇总结果

## 关键技术细节

### 数量处理规则

不同型号的"数量"字段含义不同：

1. **编号中数量是总数量**（不需要乘CSV数量）：D4, D5, D11等
2. **编号中数量是单件数量**（需要乘CSV数量）：D6-D10, D12-D16, Q3, U1等

测试脚本中通过 `models_needing_csv_qty` 列表控制。

### 钢板计算

钢板设计量使用`plate_size`字段（如"200x300"）而非固定宽度：
- 解析`plate_size`获取实际宽度和高度
- 部分型号（如D18）使用固定宽度（10.5mm）

## 开发指南

1. 在 `hgttools/data/supports_db.py` 中添加型号配置
2. 在 `hgttools/services/parser.py` 中添加解析方法 `_parse_xxx`
3. 在 `hgttools/services/calculator.py` 中添加计算方法 `_calc_xxx`
4. 在 `test_with_original_data.py` 中添加测试用例


## 已知问题

1. **重复零件匹配**：当同一管架有多个相同名称+规格的零件时（如D15的两组角钢），测试脚本无法区分
2. **精度差异**：部分型号（如D16）存在微小精度差异（<1%），可能与原始Excel的系数有关
3. **型号覆盖**：尚未验证所有型号，部分新型号可能需要调整计算逻辑
